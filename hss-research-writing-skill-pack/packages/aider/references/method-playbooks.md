# Method playbooks for HSS materials

按材料类型和研究类型选择性使用，不需要每次全读。

## A. 文献综述 / 研究述评 / 史学综述

### 你要回答的不是“谁说了什么”，而是：

- 这个领域真正的争论轴线是什么？
- 看似在争论同一个问题的文献，实际上是不是在谈不同对象？
- 结论差异来自定义、方法、材料、时间段，还是价值立场？
- 哪些文献经常被并列引用，但根本不可直接并列？
- 被系统忽略的对象、尺度、时期、地区或群体是什么？

### 好的输出应包含

- 争论图而不是作者流水账
- 概念边界说明
- 证据与方法差异
- 你对争论现状的综合判断
- 下一步研究空间

## B. 历史 / 档案 / 思想史 / 概念史研究

### 对每份一手材料都要追问

- 谁写的 / 生产的？
- 写给谁看？
- 为什么在那个时间节点产生？
- 它是记录、命令、辩护、宣传、汇报还是申诉？
- 有哪些沉默或回避？
- 这个文本是“事实窗口”，还是更像“立场痕迹”？

### 常见分析动作

- 互证：用不同来源互相校验
- 语境化：把文本放回制度、事件、人物网络里
- 分期：找出关键断裂而不是机械分阶段
- 概念漂移：同一词在不同年代的含义变化
- 受众分析：文本的真正对象是谁

### 要警惕

- 以后见之明替代历史语境
- 把档案文本当作透明事实
- 只看能支持论点的材料，不看“不顺”的材料

## C. 访谈 / 田野 / 民族志 / 课堂观察

### 基本动作

1. 开放编码：尽量贴近原话，不急着上理论标签
2. 轴心聚类：把相近的开放编码聚成主题
3. 负例分析：刻意找和主主题不合的材料
4. 位置性 memo：研究者自身位置对材料的影响
5. 场景复原：发生在哪里，谁在场，什么氛围下说出的

### 你需要的不只是“主题”，还包括

- 主题之间的关系
- 主题为何出现
- 哪些话是规范性表演，哪些是经验陈述
- 哪些沉默、迟疑、笑场、改口值得写

### 引语选择原则

- 不是挑最顺口的，而是挑最能推动分析的
- 典型与偏离都要保留
- 匿名与敏感信息必须处理

## D. 政策 / 法律 / 判决 / 制度研究

### 重点看什么

- 主要行动者是谁
- 文本设定了什么目标
- 通过什么工具推进
- 对谁施加约束 / 提供资源 / 划定资格
- 执行机制在哪里
- 问责机制在哪里
- 与上位法 / 旧政策 / 实践执行之间有什么缝隙

### 适合建立的比较维度

- 目标
- 工具
- 执行主体
- 对象
- 激励 / 惩罚
- 监督
- 语言框架
- 时间变化

### 常见陷阱

- 把法律文本的规范意图当成现实效果
- 只读条文不读实施细则 / 判决理由 / 配套机制
- 不区分“纸面制度”和“执行制度”

## E. 媒体 / 平台 / 传播 / 话语分析

### 建议关注

- 框架（framing）
- 标签与命名
- 叙事角色分配
- 责任归因
- 情绪动员方式
- 沉默对象或不可说之物
- 不同时段的话语转换

### 可以做的比较

- 不同媒体之间
- 同一媒体不同时期之间
- 官方文本与媒体文本之间
- 主流媒体与边缘平台之间

## F. 定量 / 混合方法写作（HSS 语境）

### 至少说明

- 数据来自哪里
- 变量如何定义
- 样本有什么边界
- 模型或识别策略的逻辑是什么
- 结果到底支持什么，不支持什么
- 与定性材料如何互相补强或互相制衡

### 要避免

- 用“显著”代替“有意义”
- 用表格替代解释
- 从相关直接跳到因果
- 只汇报主结果，不写稳健性边界

## G. 理论 / 哲学 / 概念分析

### 可用动作

- 定义分解：一个概念拆成哪几个必要部分
- 边界测试：什么例子应纳入，什么例子应排除
- 谱系回溯：这个概念的历史层累是什么
- 反例施压：哪种情形最能挑战你的定义
- 相邻概念区分：它和邻近术语到底差在哪

### 好的理论写作应该有

- 可工作的定义，而不是只会发光的定义
- 对常见反对意见的预处理
- 可回到经验世界的解释力
